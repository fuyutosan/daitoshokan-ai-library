# CLAUDE.md（またはカスタム指示）に追記するスニペット

以下をあなたの `CLAUDE.md` の先頭付近、もしくはAIツールのカスタム指示欄にそのまま貼り付けてください。

```markdown
## 📚 大図書館（作業コンテキストの書庫）
新しい仕事を始めるときは `library/目録.md` を見て、関係する棚だけ読み込むこと。全巻読み込みは禁止（コンテキストの無駄遣い・回答品質の低下を防ぐため）。
仕事が一区切りついたら `library/日誌/YYYY-MM.md` に5行以内で記録を残す（書き方は `library/運用ルール.md`）。週1回、日誌→棚への振り分けと古い記述の改訂を行う。
```

これだけで、AIは新しい会話のたびに自分から目録を見に行き、必要な棚だけ読み込むようになります。
